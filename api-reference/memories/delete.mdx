---
title: "Delete memory"
description: "Remove a memory permanently using its ID."
---

<ParamField header="Authorization" type="string" required>
Bearer token used for authentication. Format: `Bearer YOUR_API_KEY`.
</ParamField>

<ParamField path="memory_id" type="string" required>
Identifier of the memory to delete.
</ParamField>

<RequestExample>
```bash cURL
curl -X DELETE "https://api.flumes.com/v1/memories/mem_8dfe6a4f" \
  -H "Authorization: Bearer $FLUMES_API_KEY"
```
</RequestExample>

<ResponseExample>
```json Success
{
  "message": "Memory deleted successfully!"
}
```
</ResponseExample>

<Warning>
Deletion is irreversible. Ensure you no longer need the memory before calling this endpoint.
</Warning>
