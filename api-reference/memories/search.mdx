---
title: "Search or list memories"
description: "Retrieve recent memories or perform semantic search within a user, agent, or run scope."
---

<ParamField header="Authorization" type="string" required>
Bearer token used for authentication. Format: `Bearer YOUR_API_KEY`.
</ParamField>

<ParamField query="user_id" type="string">
Filter memories belonging to a specific user or session.
</ParamField>

<ParamField query="agent_id" type="string">
Filter by agent identifier. At least one of `user_id` or `agent_id` is required.
</ParamField>

<ParamField query="prompt" type="string">
Prompt text to run semantic similarity search against your stored memories. If omitted the endpoint returns the most recent memories.
</ParamField>

<ParamField query="limit" type="integer" default="20">
Maximum number of results to return (1-200).
</ParamField>

<Tabs>
<Tab title="List latest memories">
<RequestExample>
```bash cURL
curl -X GET "https://api.flumes.ai/v1/memories?user_id=session-123&limit=5" \
  -H "Authorization: Bearer $FLUMES_API_KEY"
```
</RequestExample>
</Tab>

<Tab title="Semantic search">
<RequestExample>
```bash cURL
curl -G "https://api.flumes.ai/v1/memories" \
  --data-urlencode "user_id=session-123" \
  --data-urlencode "prompt=What is the capital of France?" \
  -H "Authorization: Bearer $FLUMES_API_KEY"
```
